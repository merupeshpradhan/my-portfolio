# 📂 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/merupeshpradhan/my-portfolio
```

Move into the project folder:

```bash
cd your-repository-name
```

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open in browser:

```bash
http://localhost:3000
```
